Mapper with Map Hub

This macOS preview connects Mapper to the COC map library and includes the current tiled-imagery work.

  • Download for macOS — Apple silicon (.dmg)

Ask Ethan for a COC map library access token.

  1. Install Mapper from the disk image.
  2. Open Mapper and click Map Hub.
  3. Enter the access token in the first-use connection screen.

This preview is unsigned. If macOS blocks the first launch, Control-click Mapper in Finder, choose Open, then confirm.

Version 0.9.7-COC.5 · packaging refresh July 22, 2026

Mapper (COC Base)

This upstream-faithful build adds online satellite/aerial imagery and importable imagery catalogs. It does not include Map Hub.

  • macOS — Apple silicon disk image (.dmg)
  • Windows x64 installer (.exe)
  • Windows x64 portable build (.zip)
  • Linux x86-64 portable AppImage
  • Linux x86-64 Debian/Ubuntu package (.deb)
  • Linux x86-64 RPM package (.rpm)
  • Linux x86-64 Flatpak bundle
  • Android arm64 preview (.apk)
  • Release manifest (.json)
  • Puget Sound imagery catalog (.oic)

The macOS and Windows packages are unsigned. On macOS, Control-click Mapper and choose Open if prompted. In Microsoft Defender SmartScreen, select More info, then Run anyway.

The Android APK is preview-signed for direct installation rather than Play Store distribution. Uninstall an older COC preview first if Android reports a signing conflict.

  1. Open Templates > Online Imagery.
  2. Paste a tile URL or import the puget-sound.oic catalog.

Source

The Map Hub preview is built from the public COC preview branch. Mapper (COC Base), its packaging, and exact release source are preserved in this repository. Both are available under GPLv3.
