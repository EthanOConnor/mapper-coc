Version 0.9.7-COC.5

OOM — COC Edition: Base

A Windows preview build of OpenOrienteering Mapper with online imagery support.

Download

  • Windows installer (.exe)
  • Portable build (.zip)
  • Release manifest (.json)
  • Puget Sound imagery catalog (.oic)

Windows x64 only.

The installer is unsigned. In Microsoft Defender SmartScreen, select More info, then Run anyway.

Imagery quick-start

  1. Open Templates > Online Imagery.
  2. Paste a tile URL or import the puget-sound.oic catalog.

Source
